Nexus Competitivo

https://nexus-competitivo.vercel.app/

Sistema Web SPA (Single Page Application) desenvolvido como Atividade Avaliativa Final para a disciplina de Programação Front-End da UniCesumar (Campus Ponta Grossa).

Descrição
O Nexus Competitivo é uma plataforma de cadastro e gerenciamento de perfis de jogadores de esportes eletrônicos, com foco em Valorant e Rocket League. O sistema realiza operações completas de CRUD (Criar, Ler, Atualizar, Deletar), garantindo a persistência dos dados no `LocalStorage` do navegador, com validações de segurança contra duplicatas. Além disso, a aplicação consome uma API externa assíncrona (Agify) para processar os dados inseridos e estimar o nível da conta do competidor baseando-se em seu nickname.

Tecnologias Utilizadas
* **React.js** (Componentização, useState, useEffect)
* **React Router DOM** (Navegação dinâmica de rotas SPA)
* **Tailwind CSS** (Estilização estrutural e responsividade)
* **Fetch API** (Consumo assíncrono de JSON)
* **Vite** (Ferramenta de Build)
